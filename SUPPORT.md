# Support

## Scope

This repository provides organization-wide guidance for support across blackoutsecure
projects. Individual repositories may define additional support channels or SLAs.

## Getting Help

- Review the README for project context.
- Open an issue using the available templates.
- For security concerns, use the process in SECURITY.md.

## What to Include

- Clear description of the problem or request.
- Steps to reproduce or expected behavior.
- Environment details and relevant logs.
- Expected impact or urgency.
- Links to related issues or pull requests.

## What We Can Help With

- Clarifying intended behavior
- Investigating suspected defects
- Reviewing proposed changes

## What We Cannot Support

- Password resets or account recovery
- Production operations owned by third parties
